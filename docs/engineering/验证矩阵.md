## 4. 当前已知风险

- 浏览器 e2e 已建立；当前 smoke 运行时保留既有 React/Semi UI console warning，后续 UX/accessibility 阶段再治理。
- `npm audit --audit-level=high` 当前通过，但报告 2 个 moderate 漏洞：
  - `ajv <6.14.0`：使用 `$data` option 时存在 ReDoS 风险，建议在依赖治理任务中评估 `npm audit fix` 对 lockfile 的影响。
  - `brace-expansion <1.1.13`：zero-step sequence 可能导致 hang 和内存耗尽，建议在依赖治理任务中评估 `npm audit fix` 对 lockfile 的影响。
- Vite 构建当前通过，但提示 `lottie-web` direct eval 和主 chunk 过大；Phase 4.1 已记录构建基线：最大 JS chunk `index-BQPQ4Hqj.js` 为 16271.88 KB，最大 CSS `index-uUgFFXzo.css` 为 404.78 KB，JS/CSS 总量 17044.63 KB。Phase 4.3 后 SQL parser 已拆成独立 chunk，最大 JS chunk 降至 `index-CD-l3hj7.js` 13682.29 KB，`node-sql-parser` 独立 chunk 2384.28 KB，`oracle-sql-parser` 独立 chunk 168.07 KB。Phase 4.4 后 image/PDF export libraries 已拆成独立按需 chunk，最大 JS chunk 降至 `index-BqVYr10S.js` 13279.33 KB，`jspdf.es.min` 独立 chunk 390.17 KB，`html2canvas` 独立 chunk 194.90 KB。Phase 4.5 后 Landing tweet/social widgets 已延迟加载，最大 JS chunk 降至 `index-c__2pYW3.js` 13248.74 KB。Phase 4 退出门禁复核时最大 JS chunk 为 `index-Dony_C59.js` 13250.51 KB，最大 CSS 为 `index-CqULjIum.css` 390.89 KB，JS/CSS 总量 17023.50 KB，仍在预算内。
